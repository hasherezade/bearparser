Portable Executable parsing library<br/>

Requires:
+ Qt5 SDK<br/>
+ Qt5 Core<br/>
+ cmake http://www.cmake.org/<br/>

<b>Possible to build with Qt4 (instead of Qt5) by setting a CMake flag: USE_QT4</b><br/>
Example:
```
cmake -G "Unix Makefiles" -D USE_QT4=ON ../
```
